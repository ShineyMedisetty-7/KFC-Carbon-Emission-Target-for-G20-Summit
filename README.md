## 📖 About the Project

This project was developed as a collaborative team project to support carbon emission monitoring and sustainability analysis during the G20 Summit. It provides a secure and user-friendly survey platform with real-time analytics for administrators.

---

## ✨ Features

- 🔐 Secure Authentication (Email, Phone & Google OAuth)
- 📋 Interactive Carbon Emission Survey
- 📊 Real-time Analytics Dashboard
- 📈 Charts and Data Visualization
- 📱 Fully Responsive UI
- 🔒 Supabase Authentication & Row-Level Security
- ⚡ Fast Performance using Vite
- 🎨 Modern UI with Tailwind CSS & Framer Motion

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/KFC.git
cd KFC
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file:

```env
VITE_SUPABASE_URL=YOUR_SUPABASE_URL
VITE_SUPABASE_ANON_KEY=YOUR_SUPABASE_ANON_KEY
```

### Start Development Server

```bash
npm run dev
```
